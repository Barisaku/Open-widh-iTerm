Open widh iTerm
====

macのFinderから現在Finderで開いているカレントディレクトリをiTermで瞬時に開く事ができるアプリです。


## Description

このアプリをFinderのツールバーにセットし、クリックすると現在Finderで開いているカレントディレクトリをiTerm上で瞬時に開くことが出来ます。

## Usage

1. Finderのツールバーにセットした `Open with iTerm.app` をクリックします。
2. 
  - iTermが起動していない場合は、iTermの新規ウィンドウにカレントディレクトリが表示されます。
  - iTermが既に起動している場合は、iTermの新規タブにカレントディレクトリが表示されます。

## Install

1. `Open with iTerm.app` を `~/Applications` フォルダに移動。
2. commandキーを押しながら、`Open with iTerm.app` をFinderのツールバーにドラッグ&ドロップします。
3. ドロップ&ドロップしたアイコンをクリックするとFinder上のカレントディレクトリをiTermで開きます。
